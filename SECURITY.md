# Security Policy

## Reporting a vulnerability

If you discover a security issue, please report it responsibly:

1. Do **not** open a public issue for security vulnerabilities
2. Contact the repository owner privately

## Privacy

WebProbe is a static client-side application:

- No backend, authentication, or database
- No analytics or telemetry in the MVP
- Browser information and benchmark results stay local unless the user copies them elsewhere

The **My Browser** page exposes information your browser already makes available. Avoid sharing it in contexts where fingerprinting is a concern.

## Scope

In-scope: vulnerabilities in WebProbe code that could affect users visiting the hosted site.

Out-of-scope: browser engine bugs, third-party linked resources, and issues in dependencies without a demonstrable impact on WebProbe.
